# BackHole-Drive
The following game simulates a ship orbiting aroud the inner solar system using a blackhole drive

The ship uses a blackhole drive, it takes profit Hawking radiation of the balckhole to create two beams that can be used as thrust. 
Because of the constant evaporation of the bakchole the mass flow output is constant and as the blackhole evaporates its Hawking radiation power increases.
If the power exceeds x2 nominal power the ship is destroyed. 
The aim of the game is collenting mass from asteroids and feed the blackhole for as long as posible to avoid a runaway.
